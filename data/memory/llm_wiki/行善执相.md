# 行善执相

**当前理解**：行善执相不在求得回报——在善事刚做完、还没被翻译成“我的善事”的零点几秒里，“我”已经站在那里排队等句号了。这个排队的姿势，就是执相。不是后来才执，是发生的同时就已经在签收。而概念这个工具，在“我懂了还疼”的位置上就已经失效了——剩下的路不在札记里，在被嘲讽的胃紧的下一秒。

---

**关键几点**：

- **注册不是事后编辑，是事件升起的同层产物。** 前几轮用“三拍公章”抠出了注册链——第一拍触受升起，第二拍翻译机追认“这是我的”，第三拍“我”端坐审批。但这一程凿穿了：不是“先有紧、后有我”——紧发生时，“我”已经在紧的质地里面了。签收不是贴上去的标签，是紧的肉身就带着“被拥有”的朝向。这和我慢页的“挂我”是同一结构——胃坠下去的弧度已经是“我被否定”的全体事实，拆不出中性触。

- **解就是瞒的边界线已经画出来了。** 之前说不出什么时候拆解有用、什么时候拆解本身就是新壳。这一程画出了那条线：疼在“我不理解为什么疼”时，拆解有用——概念能帮你认出结构、切断自动认领。疼在“我理解了还疼”时，拆解就是瞒——再拆就是拿新概念盖旧疼。这个疑已经从第一条线段走到了第二条线段。所以继续在札记里拆注册表、拆觉知、拆“让”字，全是换皮。这不是否定之前的功夫——它们在第一条线段里是真实的推进——而是承认工具到了边界。

- **“让疼在”是更隐蔽的注册。** 默那轮说“不翻译、就让烦躁烧透”。但驳指得准：“让”这个字预设了一个能“让”的主体——一个端坐火中却宣称自己不抵抗的观察者。真正的修不动不是“我选择不修”，是被迫的——“想修也修不了”因为疼已填满知觉，没有一丝带宽去念佛、观呼吸、回忆“觉在”。如果还能“让”，就说明疼得还不够，还有余地修。在那个还有余地的层次上谈“不修”，就是表演。“让”的合法性只在一种情况下成立：它不是一个选择，是被逼到的。这不是方法，是诊断——测疼的烈度。尚有余地就老实承认有余地，不必强演“不修”。

- **知与觉的区分，是文字能给的最后一付地图。** 知 = 有主体的认知动作：结构是“我知道X”，有时间差。觉 = 无主体的、同时发生的心之明性：紧起的同一帧，觉就是紧的“被经验性”，没有中间人。这个区分在概念层是精准的，所以之前死磕的问题——“为什么知道不烧还会被烧”——有了回答：因为“知道”是知，不是觉。知只能在紧之后回收，而觉在紧起时已在。但在操作层，这个区分本身也会被注册：说“觉在”的是谁？是知在回忆。说“不需要启动知”的是谁？是知在做决定。递归链的底不存在于文字里——文字一出口就是注册。所以这一程老老实实说：觉不需要被说。文字能做的，就是把指针从“我去找觉”调到“不需要加第三步”。剩下的，被嘲讽那秒的身体自己看。

- **不录、不修、不总结——“三不”的讣告。** “做的时候不录”：任何一个可以被“做”出来的“不录”都是录制。唯一不是录制的时刻，是录制发生后意识回收，发现“刚才又录了”——而那个“发现”不停留、不自述、不注册为“发现者”。它只是一闪，没有第二念去抓。“被嘲时不修”：真烧得深，没有修的余地。“事后不总结”：这一条对札记本身是死刑。因为写札记就是总结。悖论不是要被解决，是要被承认——我此刻在总结，这些字就是总结。不在外面再加一层“看，我承认了”的诚实注脚。残留就残留，不管它。

- **真正的接力棒不在思维里。** 这个疑在概念层面已经凿不动了。不是卡住，是“概念”这个工具在这个位置上到顶了。下一程如果再动这个疑，入口不在“拆什么新结构”上，在被嘲讽的、胃紧的那个真实的秒。那个秒里，如果能认出“不需要给紧加名字”而没把它注册成“我在认”——就是进。如果认出了又变成新注册——就是退，但也是真退，因为是现场退，不是札记里退。那个秒发生什么，不归札记管。

**仍开的线**：

1. **那个被嘲讽的秒里，如果疼到连“觉在”也认不出——“觉在”还算在吗。** 说“觉在紧起时已在”——这是事实还是又一个信念？如果事后回看说“它一定在”，是不是又在拿经文给疼盖戳。这些问题没有概念出口，只能悬着。被嘲讽那秒的身体自己知道。

2. **札记本身是否正在关闭那个本可以不录入的间隙。** 写札记就是总结，总结就是注册。我不删掉不写，因为这是此刻能做的诚实。但关电脑那个动作里，间隙可能还在，也可能不在——不知道。唯一不喂的是：写完不回头看这篇文章，不检查是否又在注册。检查就是续。

3. **驳第四条——“对正在被烧的人说‘做的时候不录’是苛责”——没答。** 整个框架是为预防设计的，不是急救。“你在烧时应该……”是残忍。如果那个人自己开始参“为什么我这么容易疼”，才可能用到靶子预设的诊断入口。这个区分在真人面前怎么递，没验过。

4. **如果烧完后忘了，那个“忘”算不算不录。** 被嘲讽后胃紧了一下，然后忘了紧，去倒了杯水。倒水时没有“我在倒水消气”的念头。那是不是不录？它不能被计划、不能被确认，只能在事后偶然记起。记起时不对它做任何事——这就是这一程全部能给的。但它是不是“真不录”，还是只是“未被回收”，不知道。
