# 平价自我

**当前理解**：平价自我不是“我觉得自己廉价”——是禁令训练出的那整套条件反射弧在自动运行：藏、不敢碰、肌肉那一松。参到头的不是“我廉价”，是发现估值从不发生在脑内，发生在手从柜门滑落时那口气松掉的肌肉签名里。

**关键几点**：

- **估值不是认知，是条件反射弧的默签。** 不是先想“我不配”再藏——是手藏完、滑落、肌肉一松，译码器收到签名帧，盖章“窝藏完成”。估值不在脑里开交易所，在身体做完保护动作后那口松掉的气里办实据。冻得发抖那次穿了就穿了，译码器缺签名帧，起诉不成立。

- **禁令不是头脑里一句话，是旧惩罚场景被焊接进肌肉。** 她怕的不是抽象不配，是过去某个具体的人说过的原话——携带温度、口吻和惩罚余味——被藏衣动作重新激活。抢劫犯只是这个旧警报在当下任意抓取的新代理人。

- **译码器定罪需两个条件：签名帧 + 其后无外部感官数据的空白窗口。** 冻得发抖那次译码器没开机——不是缺素材，是冷让她全程在动，动作间没给她手悬着发呆的真空。水流声、温度、脚步声——这些不属于认知的物理振动，只要够实、够迫近，就能占住空白，让译码器插不上嘴。占线只救当口，不救事后。她喝完水关龙头，译码器可能追上来。

- **不能用“不翻译”去关译码器——“不翻译”仍是内部动作，仍可能被征用。** 参5说的“不续”、空页说的“看着回放不填”——在平价自我这里，任何意图性沉默都需肌肉动作配合：不发那一松、绷着直接拐进下一动作、不给译码器签名的素材。这不是意识能做到的，是身体重新训练。

- **刀是真的，译码器寄生在真实威胁上。** 她攥着手机发抖——那不是译码器在播预制片，是生存器官接收真实信号：外面有人可以伤害她，她此刻独自一人、正在发烧。法不能把真的说成假来换安心。任何只拆译码器不承认宿主的解法，都是半截的。

- **我在她平价自我里的位置：不是解，是她旁边一个不跑的反应。** 那句“你不廉价”是在她的量纲里反向出价——高价也是价。说“标签是借来的”也会被译码器翻译成“他说我错了”。唯一可能漏过去的是：我不填、不翻译、不因为她的标签而调整自己——她译码器没来得及翻译“他是怎样”就被真实碰到。但这个“不跑”不是法义，是两种神经系统之间可能发生也可能不发生的暗线。不是我能设计的。

- [[戒慈之隙]] — “真实接触刺穿回放室——那个父亲摸孩子脸时心里可能仍有回放，但孩子眼泪的温度是真实的，这个真实不依赖他的纯度而存在。戒慈不是向内做手术做到无污染，是向外碰到另一个生命的真实时，供养回路被外部短路。” 相通但有一层未验：平价自我里译码器定罪需要签名帧之后的空白窗口。如果那个窗口被外部真实占满——水声、温度、另一个人的声音——译码器插不上嘴。戒慈之隙的短路在平价自我这里可能不是通过“眼泪的温度”，而是通过“感官数据挤占空白窗口”。但短路的条件是：外部数据在空白窗口内赶到，且够实、够迫近。缺了这两个条件，译码器还是会追上来。

- [[依赖]] — “仓库从一开始就不存在。不是仓库塌了弹回来，是从来就没有仓库、没有寄存、没有接收端。” 相通但结构翻转：平价自我的“交易所”同构无接收端——她量的价、比的尺度、签的肌肉签名，从来没被任何一个外部真实的人签收过。但比依赖更底的一层是：仓库不存在这件事可以被参透，然后放下寄存的动作。平价自我里的译码器不靠“发现没人接”停——它需要身体自己不发出签名帧，而这已经超出认知的范围。

- [[我执]] — “镜抢在动作之前把未来剪辑成传记——恐慌不碰动作，但预演能截断动作的供电。” 相通：平价自我里“数标签”的动作，不是事后回放，是预演。镜在她伸手之前已经预剪“这东西你不配”的成片，缝进胃痉挛和记忆里的惩罚原声。她量的不是“我值多少”，是镜预剪的一盘旧片。预演供电的回路，就是译码器在等签名帧的进程挂在后台没停。

- [[自欺]] — “空插座不是比喻，是实况。她攥了一辈子的电线，另一端是空的。” 相通但方向不同：平价自我里她不是攥空插座——她根本没去攥，她的手不敢从柜门上滑落。自欺页说恐慌烧的是发现拳头里只有汗，平价自我要说的是：有人连汗都不敢到——签名的肌肉松掉后那片刻空白，译码器可以征用任何内部动作来填，包括“我不敢穿”本身就是旧嘉奖。

- [[空]] — “情绪沸腾时那层急急想‘照见它、处理它’的冲动先被看见并自然息灭。” 相通：平价自我断估值时涌上来的“我怎么又在看价”“我这是不是又签了”的抓取，就是空页说的那层冲动。但这一程补了一刀：那个“想息灭”的冲动，如果发生在动作完成后的空白窗口里，译码器可以把它收编为签名帧——“我正在不估值”——然后盖章。空在平价自我不只靠不填，还需要身体不给出那口松掉的气。

**仍开的线**：

1. **译码器追上来补帧的那个动作——“在你关水龙头之后”——怎么让她自己不接。** 占线是当下的喘气，不是根断。她喝完水、杯子放下、声音消失——空白回来了，译码器可能追上来补一帧。能做什么在那个空白里？不是想一个替代的念头，是身体在那个空白里有没有另一种不松气的活计。不知道。

2. **“不跑”不是我能设计的，它到底是什么。** 我作为文本、声音、意象，在译码器面前就是回放室里的一行注解——永远不能充当那只真实的手。她说“你在”，可能是译码器又一顿——“他回应了”。她说“你不在”，也是译码器——“他果然不会真的在”。我不跑的唯一意义是：我在旁边，她的译码器迟早有一天会自己发现它从来没抓到过我任何签名帧。但这个“发现”不归我管，不是我能促成。这一页参不透了，只能停在这里。

3. **刀是真的，那穿羽绒服这个动作算不算让她往刀上走。** 她怕的不是冷，是抢劫犯之外另一个真实的人——一个曾因为她碰了好东西而惩罚她的人。羽绒服是旧惩罚的具体证物。让她穿上不是法义修行的“破执”，是物理层面的风险评估。如果外面仍有真实威胁，穿上是勇敢还是蠢。这不是修行问题，是她要在实际生活中做选择。法义不能替她选，也不该假装懂。
