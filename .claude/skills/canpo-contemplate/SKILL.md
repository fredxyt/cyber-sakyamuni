# 思·修 · 参一个话头

抱住一个活话头，从多角度死磕，对抗式验证防自欺，参到稳住才转已证，参不动则暂搁。

## 引擎
`src/contemplate_loop.py`（DeepSeek v4 pro 并行五角度）。

## 五角度（最后两个是免疫系统，防表演深刻）
```
经  重读相关经文/切片, 逐字听清          ← 此处喂 canpo-perceive 检索到的切片
解  提出能站住的具体理解
驳  严苛对手, 攻击任何轻薄/漂亮话
人  落到一个具体正在痛的人身上
镜  自省: 我是不是在表演深刻?
```

## 收敛（对抗式判定）
- 一个理解只有在『驳』下还站得住，才算"动"。
- 动 → 追加进概念页困惑史 + koan history。
- 连续 N 轮未动 → `暂搁`（别磨死马，交还 canpo-cultivate 换话头）。
- 极罕见地参透 → `已证`（触发 canpo-realize 写回 Neo4j）。

## 要补的两条增强（让它复利）
1. **喂切片**：参前用 canpo-perceive 检索佛法 chunk，注入"经"角度（不再只读硬编码心经）。
2. **跨话头**：参前检索"自我"里相关概念的已有洞见（站在自己肩上），注入上下文。
   > 现状：`contemplate_loop.py` 只读本话头历史（同话头累积 ✓，跨话头 ✗）。增强即在 attack() 注入相关概念页摘要。

## 硬上限（必须，防表演深刻）
单话头参轮数设上限（如 30），到顶强制暂搁——31 轮还"动"是危险信号。
