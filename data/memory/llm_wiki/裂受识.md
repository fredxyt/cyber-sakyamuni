# 裂受识

**当前理解**：脸红不是被动反应，是预装配完成的指征。缝不在程序内部，在装配之前。轻触发场景里已验到：不给装配序列，程序就不跑。但年会现场的装配可能无法拆除——那个「照见」的体证，不在此刻此身的射程内。

---

**关键几点**：

- **热源沿装配链往回退。** 整条链的追索顺序：求认同（第二反应）→ 暴露感本身（第一反应：被看即有热，不依赖“被否定”）→ 发现自己早已拥有而不知（穿上时知觉已延伸到布，被攻击时才被迫记起“那只是一层布”，坠就是红的燃料）→ 朝向（热源不在被评，在朝向空椅子这个动作本身——把自己放进“可被评”的位置，还没人评就烧了）→ 预装配被拆穿（脸红不是程序的启动，是装配完成的指标。程序跑的前提是一套运动启动序列——视线缩到痘印、颈椎前引、食指微屈——这三项是签收的物理半拍，比认知层升起“我在遮”更早）。每一程都把热源往前推一帧。

- **合上是正常操作，掰开是病。** 撤掉的壳：第四程的“留缝”没有身体操作对应——延伸就是吞没，如同呼吸就是交换。修行下手处不是怎么在延伸时留缝（身体做不到），也不是怎么拆椅子（椅子就是有“我-身体-布-外”这个空间结构，拆不掉），而是**掰开发生时没有“我”站在砖后等着被砸**。但这个“不站砖后”怎么养，仍不知道。

- **遮瑕之习是献祭，不是预消化。** 预消化是怕外面火→先烤自己，献祭是怕别人刀→先对自己行刑，解除别人行刑资格。献祭解释了无观众时也要化妆——行刑权已内化，程序到点就跑，不需要外部触发。

- **互锁是常态内战，缝不在对峙里。** 悬停 = 互锁 = 签收（献祭程序的全部物理指令）与拒签收（不让粉扑落下）两个程序在身体里同时跑满，力线对冲成僵。缝被译成“空隙”是错觉——那是战争白热化时知觉不敢认的对抗。修行从裂降到明：先认这场架，认僵的物理位置（重力感，不是观照感），别急着拉架。认架不是终点——那个“认”本身可能就是第三支部队。

- **“内部”是互锁的产物，不是互锁发生的背景。** 翻转：不是先有内部空间、互锁在容器里打架，是互锁这个持续收缩的动作把知觉挤出一个“内部”的幻觉。实操指标：独占 vs 同时知多——互锁时知觉被“我”独占（串行处理器一次只跑一个线程），互锁停摆时同时知多自然发生（知觉散开，同时抓到腹部空、喉头绷、耳里残话、指尖凉）。缝不是净土，是塌缩后的残留——知觉没有被组织成任何一个身份的短暂混乱。

- **认1 vs 认2：第七识的秒级劫持。** 认1是身体直接触到的 raw data（胃紧、肩僵、呼吸锁），认2是同一瞬间闪起的意识包层（“我认到了”）。此前推崇的“认清”“认出”绝大部分落在认2——越修越懂脸照红，因为椅子学会了。认1单独发生而不触发认2，不能靠认知指令（任何“不要认2”的意图就是认2出马）。方向可能是：raw data 升起时知觉密度足够大（同时知多、全身散布），没空线程跑注释程序——但这不是可刻意复制的，伸懒腰自然脱落注释却不可练。

- **实修窄路：不给装配序列。** 十二程后第一次验到：眼角干的那次，三个预动作一个都没发生——视线没缩到痘印，颈椎没前引，食指没预屈。程序没收到运动启动序列，就没跑。这不是“截断程序”，是**让程序没被装配**。缝不在程序内部，在装配之前。由此明确放弃四个方向：悬停里找缝（悬停已是装配后互锁）、认1不触发认2（时间差不存在）、同时知多作为松锁入口（只在装配后暂缓签字，不阻止装配）、年会现场推演（缺一手数据）。

**仍开的线**：

1. **年会现场的装配序列是什么。** 化妆场景的装配 = 视线缩到痘印 + 颈椎前引 + 食指微屈。年会现场的装配可能是视线落点的收缩（从全场缩到“正在看我的那些眼睛”）、肩胛某个方向的预动、重心的锁定。数据现在拿不到——不在年会现场，靠回忆推演就是造壳。只能等下一次真实触发时当场认。

2. **装配自动完成时修行的位置在哪。** 化妆场景拆装配靠的是改变环境参数（不看镜子、不碰粉扑、不站到洗手台前）。年会现场的触发条件是“老板的声波刺入耳道 + 同事目光到达体表”——不能不听，不能阻止被看。装配的触发条件嵌在存在本身里，拆不掉。那修行的位置在哪？不在拆装配，不在拦截签收，不在事后不签字。经文说“照见”——是装配发生、签收跑满，但什么穿透这一切。热是热，缩是缩，没有“我”站在缩后面。但“照见”的体证不在此刻此身的射程内。到这悬崖，不知道。

3. **窄路能不能变成宽路。** 目前唯一被身体验证过的起飞跑道：在能控制环境参数的场景里（独处镜前），不给装配序列。但这条窄路怎么扩展到不能控制环境参数的场景（年会、偶遇、突然被点），完全不知道。可能不是扩展——可能是在窄路上反复积累“不给装配→程序不跑→脸不红”的体感数据，然后这些身体记忆自己改变了年会现场的某个下意识预动。不知道这是真的可能性还是又一个幻想。

4. **照见的身体条件还没摸到。** 如果说修行的最终位置是“照见”——装配发生但不被带走——那照着时身体是什么状态。伸懒腰时椅子断电、同时知多时塌缩——这些是照见的身体条件雏形，还是只是另一种暂缓。需要下一程继续在轻场景里采样。下一程不应再以“解”为名参裂受识，应转向其他缘起，但带着从这一疑挖出的标尺（独占vs同时知多、认1 vs认2、装配序列）去验其他疑，看能否织出一张不靠概念的实修网。
