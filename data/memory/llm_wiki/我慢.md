# 我慢

**当前理解**：我慢不是羞耻、不是愤怒、不是“护”——是翻译机在签收“我被否定了”的同时，给自己发了一张“我没签”的证明。那个自动注册的动作本身，就是我慢的种子。松脱不在不签收，在发现签收从未真正完成过。

---

**关键几点**：

- **“应激和标签分离”是假象，分离发生在触之后。** 35岁女性胃坠的那一刹，身体已经认出了“这次又是我”——标签嵌在应激里，不是事后追贴的。真正的捆在触的现场：知升起的同一瞬间，知者就被生出来了。这不是两个步骤，是一个硬币的两面。

- **护的根本不是对象，是“捆”这个动作不肯停。** 脸热、喉紧、脑白、嘴闭——身心现象的散片。心事后把它们捆在一起，贴上“我被辱了”的标签，然后守着这个标签说是“我受伤了”。守的从来不是伤，是捆。能说“我看到捆了”的那个观察者，是捆好之后才出现的产物——它的名字叫“签收单”，不叫“拆捆的人”。

- **所有“移除‘我’”的操作，都在续命“我”。** 观、不翻译、不签收——这些都在往那团疼上面放一个人。胃坠现场没有那个人。“不签收”本身也是翻译机在签收之后给自己发的一张“我没签”证明。胃坠不是“我被否定了”的符号，胃坠是胃坠。那个从坠到“我被”的半步自动翻译，不是非要跟着走。

- **“谁在感知”的追问，是翻译机在找注册对象。** 知觉本身就是标签。胃坠下去的那个弧度，已经是“我的存在被压”的全体事实，拆不出无标签的纯触。不是从触中抢出中性的热，而是允许那个带着纹路的沉完全是我们此刻的形状，不急着解释，不急着修正。死结不是被解开，是被完整看过。

- **肩提舌抵的肌肉收缩，比翻译更快。** 挂我的物理根系，在念头还没成形之前就已经绷紧了。那个比念头更快的收缩，能不能被身体自己软下来，而不经过一个“我要放下”的念——仍不知道。

**仍开的线**：

1. **肩提舌抵的肌肉收缩，能不能在身体层面被直接认出而不经过认知的注册。** 那个比念头更快的收缩，在胃坠、脸热、耳闻“你不行”的第零点几秒就已发生。它不在翻译机的射程里，不在认知的射程里——但它能被身体自己感知到吗？如果能，那个感知是不是新的注册？不知道。

2. **如果真的不挂我，那个坠着走路的人用什么感觉自己的存在。** 不再靠痛来确认活着时，那个“活”是什么——不是问定义，是问胃那边有没有别的质地。没验过。

3. **对“观”的怀疑，是否在拆解法被自身否定的同时，又变成新的“我知道工具被否定”的快感。** 这个自反性没有任何出口。承认“观可能和执共享底层”这个承认本身，下一秒就可以被吞成“至少我看清了观的问题”。没法防，不编下一口。

4. **“缝”的质地不是中立触，是带着纹路的全体事实。** 坠下去的那个弧度，已经是“我的存在被压”的全体事实。这个全体里无主也无客，但体感上就是有重量。那个重量是什么——不是“我被压”的故事，是缘起本身的重量。这个重量在经里叫什么，不知道。
